# ترتيب — ملفات البناء الكاملة

## التشغيل
```bash
npm install
npm start
```

## البناء لويندوز
```bash
npm install
npm run build:win
```

## الموجود داخل النسخة
- شاشة ويب داخل الحاسبة والويب
- أذان محلي مدمج
- اسم الأذان قابل للتعديل من الإعدادات
- ملفات القرآن ومفاتيح الجنان محليًا
- أيقونات وهوية جاهزة للبناء
